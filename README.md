# Geoguessr Clone

This app is a clone of https://www.geoguessr.com/

Using [Google Maps Platform](https://cloud.google.com/maps-platform)

## Installation

```bash
git clone git@github.com:sirAMPR/geoguessr-clone.git
cd gueoguessr-clone
npx serve
```
